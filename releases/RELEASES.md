# Dhikrtop Release Archives

This directory contains release versions of Dhikrtop for Windows.

## Current Release

### v0.1.0 (March 2, 2026)

**Features:**
- ✅ System tray integration
- ✅ Setup wizard (4-step onboarding)
- ✅ Adhkar reminders with auto-dismiss
- ✅ Settings panel with preferences
- ✅ Multi-language support (English/العربية)
- ✅ Customizable reminder intervals
- ✅ Windows startup option
- ✅ Beautiful modern UI (maroon & gold theme)

**Changes:**
- Redesigned UI with improved spacing for small form factor
- New color palette: #6a2428 (maroon) + #cf9555 (gold)
- Fixed text overlays and component scaling
- Added React Bits-inspired components
- Improved performance and accessibility

**Downloads:**
- `Dhikrtop_0.1.0_x64-setup.exe` - NSIS Installer (Windows 64-bit)
- `Dhikrtop_0.1.0_x64.msi` - MSI Installer (Windows 64-bit, alternative)

**Installation Notes:**
1. Download the `.exe` file
2. Run the installer
3. Follow the setup wizard
4. App will launch in system tray

**System Requirements:**
- Windows 10 / Windows 11 (64-bit)
- WebView2 Runtime (usually pre-installed)
- 50 MB free disk space

## Download & Install

### For End Users

Simply download the latest `Dhikrtop_0.1.0_x64-setup.exe` and run it.

### For Developers

Build from source:
```bash
git clone https://github.com/yourusername/dhikrtop.git
cd dhikrtop
npm install
npm run app:build
```
