# Dhikrtop - Smart Adhkar & Quran Memorization Aide

A browser extension that intelligently detects the best time to display adhkar (Islamic remembrances) and provides a Quran memorization aide feature.

## Project Structure

```
/dhikrtop
├── docs/              # Documentation
├── tests/             # Test files
├── src/               # Source code
├── .gitignore         # Git ignore file
└── README.md          # This file
```

## Features

### 1. Smart Adhkar Display
- Intelligent detection of optimal moments to display adhkar
- Supports multiple detection methods
- Full customization options

### 2. Quran Memorization Aide
- Context-aware memorization assistance
- Random placement verification
- Progress tracking

### 3. User Controls
- Microphone integration
- Keyboard shortcuts
- Enable/disable functionality
